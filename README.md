# Oracle Retail Sales Analytics Project

## Overview
This project demonstrates advanced Oracle SQL analytics using real-world retail
sales scenarios. It focuses on complex reporting logic, analytical functions,
and performance optimization in Oracle 19c.

## Key Skills Demonstrated
- Oracle analytic functions
- Window functions & ranking
- Time-series analysis
- Star schema design
- Query optimization & indexing

## Database
Oracle 19c

## Use Cases
- Revenue trends
- Customer ranking
- Retention & churn
- Month-over-month growth
- Performance tuning
